# Photo Gallery Android App

This is a custom Android Photo Gallery application designed to improve image management with privacy and recovery features.

## Features
- View images in All Photos and Albums
- Swipe left and right to navigate images
- Move images to Private Vault
- Recycle Bin for deleted images
- Restore deleted images
- Smooth UI

## Technologies Used
- Java
- Android Studio
- RecyclerView
- File Handling
